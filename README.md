## 🚶 Motion Detection System using PIR Sensor and ESP8266

## 📌 Project Overview
The Motion Detection System is an IoT project developed using the ESP8266 (NodeMCU) and a PIR Motion Sensor. The system detects human movement in real time and displays the motion status on a web page. An LED turns ON when motion is detected, making it suitable for security and automation applications.

## 🎯 Objective
Detect human motion using a PIR sensor.
Display motion status on a web page.
Control an LED based on motion detection.
Learn IoT-based security monitoring using ESP8266.

## 🛠 Components Required
ESP8266 NodeMCU,
PIR Motion Sensor (HC-SR501),
LED,
220Ω Resistor,
Breadboard,
Jumper Wires,
USB Cable,
Wi-Fi Connection

## ⚙ Working Principle
The PIR sensor continuously monitors infrared radiation.
When a person moves in front of the sensor, motion is detected.
The ESP8266 reads the sensor output.
The LED turns ON to indicate motion.
The web page updates automatically to display the current motion status.
When no motion is detected, the LED turns OFF and the web page displays "No Motion Detected."

## 🌐 Web Interface Features
Real-Time Motion Status
LED Status Indicator
Responsive HTML & CSS Dashboard
Automatic Page Refresh

## ▶ How to Run
Connect all components according to the circuit diagram.
Open the project in Arduino IDE.
Install the ESP8266 Board Package.
Enter your Wi-Fi SSID and Password.
Upload the code to the ESP8266.
Open the Serial Monitor.
Copy the IP address displayed.
Open the IP address in a web browser.
Move in front of the PIR sensor to test motion detection.
